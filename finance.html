Finance Manager

Self-developed tool for personal financial insights

Overview

Developed a cloud-based Finance Manager tool that integrates credit card bills, bank statements, investments, and loans to provide unified expense tracking and financial insights.

Functionalities of the Tool

  • Integrates credit card bills, bank statements, investments, and loans into one dashboard
  • Provides expense tracking, categorization, and filtering by time range or category
  • Generates financial insights and visualizations for better decision-making
  • Supports secure storage and retrieval of financial records

Technical Details

  • Backend: Python (FastAPI) for APIs and business logic
  • Database: PostgreSQL for structured transaction data
  • Data Pipeline: AWS S3 + Glue for ETL and data analytics
  • UI: Appsmith for quick, no-code dashboards and forms
  • Deployment: AWS-based (scalable, low-cost cloud infrastructure)

Future Enhancements

  • Integration of an AI financial agent for personalized insights and recommendations
  • Support for multi-user accounts with authentication and role management
  • Expansion to include budget planning, goal tracking, and forecasting
  • Potential user acquisition as a SaaS tool, with subscription-based features
