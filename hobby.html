Hobby Projects

Cloud, Data & Automation Experiments

Overview

I work on hobby projects that blend learning and creativity, including tools for the Einbürgerungstest, my personal portfolio, and a YouTube channel to share knowledge and insights.

Einbürgerungstest Practice Website

  • Extracted the full question set from the official German Einbürgerungstest questionnaire
  • Converted the dataset into JSON and translated it into English for bilingual practice
  • Built the web app using HTML, CSS, and JavaScript for a lightweight and interactive experience
  • Deployed on AWS S3 with CloudFront CDN and made available online at exam.paulmunj.com.

Einbürgerungstest Appointment Tool

  • Built an automation with Python + Playwright that scrapes the official appointment site every minute to detect available Einbürgerungstest slots
  • Deployed the service on an AWS EC2 instance, ensuring 24/7 availability and reliability
  • Sends real-time notifications via ntfy.app (sub. tag: paulmunj-ntfy) and Telegram, compatible with both iOS and Android
  • If you don’t receive a notification within 1–2 days, feel free to contact me directly to troubleshoot or confirm your subscription

Portfolio & YouTube Channel

  • Designed and built a personal portfolio website to showcase my professional experience, projects, and certifications
  • Deployed the portfolio on AWS S3 with CloudFront CDN, ensuring scalability and fast global access
  • Manage a YouTube channel where I share insights on data engineering, coding, and career growth
  • Currently developing an AI-powered portfolio agent to interactively answer questions about my work and background
